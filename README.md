🎮 Stone Paper Scissors Game (Python)
📌 Project Overview

This is a simple Stone Paper Scissors game built using Python. The game allows a user to play against the computer through an interactive graphical interface.

It is a beginner-friendly project that demonstrates basic Python logic, GUI development, and user interaction.

🚀 Features

🎯 Interactive GUI using buttons
🤖 Computer generates random choices
🧠 Game logic to determine winner
📢 Displays result (Win / Lose / Draw)
🎨 Clean and simple user interface
🛠️ Tech Stack / Tools Used

Language: Python 🐍
Library: Tkinter (for GUI)
Concepts Used:
Conditional Statements (if-else)
Functions
Random Module
GUI Event Handling
